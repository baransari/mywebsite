Website UX Analysis

Appealing Website

What is the URL of the website?
https://www.airbnb.com
Does this page use tables? If so, for what purpose?
No, this website doesn't use traditional HTML tables for layout. Instead, it uses modern CSS Grid and Flexbox for organizing content.
Does this page use CSS? If so, for what purpose?
Yes, this page uses extensive CSS for:
  • Responsive layout design that adapts to different screen sizes
  • Consistent styling of text, buttons, and form elements
  • Visual effects like shadows, transitions, and animations
  • Complex grid layouts for displaying property listings
Is this page appealing or unappealing? List three reasons for your answer.
This page is appealing because:
  • Clean, minimalist design with ample white space makes content easy to read and navigate
  • High-quality images are prominently featured, creating visual interest and showcasing properties effectively
  • Intuitive navigation with clear categorization helps users quickly find what they're looking for

Unappealing Website

What is the URL of the website?
https://www.craigslist.org
Does this page use tables? If so, for what purpose?
Yes, this website uses HTML tables for both layout and organizing listings. The main page structure and the categorization of classified ads are arranged in table format.
Does this page use CSS? If so, for what purpose?
The website uses minimal CSS for:
  • Basic text styling (font size, color)
  • Simple layout structure
  • Link styling
However, the CSS implementation is very basic and lacks modern design practices.
Is this page appealing or unappealing? List three reasons for your answer.
This page is unappealing because:
  • Outdated design with a dense, text-heavy layout makes information difficult to scan
  • Lack of visual hierarchy - all links and text appear with similar prominence, making it challenging to identify important elements
  • Minimal use of color, images, or modern design elements creates a utilitarian appearance that feels dated
If this page is unappealing, what would you do to improve it?
To improve this website, I would:
  • Implement a modern, responsive design that works well on all devices
  • Add intuitive navigation with dropdown menus and search filters
  • Introduce a clean visual hierarchy with proper spacing, typography, and color to differentiate sections
  • Incorporate imagery where appropriate while maintaining fast loading times
  • Modernize the listing view with cards or grid layouts that include thumbnails
